# Integration

Copy `templates/` into `resources/product_templates/batch_3/`.
Merge `index.json` into your global `visual_dna_index.json`.

Confirmation loop order:
1. Vision AI extracts furniture DNA.
2. Matcher searches batch indexes.
3. Proposal builder displays closest HomeU product template.
4. User correction updates confirmed JSON.
5. SVG preview and DXF are generated from confirmed JSON only.
