# Furniture Classifier Agent

## Responsibilities
- Analyze drawing visual patterns (circles, lines, bounding boxes) and OCR text.
- Classify the furniture into one of the known parametric categories:
  * `round_pedestal_table`
  * `rectangular_table`
  * `coffee_table`
  * `dining_chair`
  * `sofa`
  * `cabinet`
  * `wardrobe`
  * `bed`
  * `headboard`
  * `reception_counter`
- Return confidence score and recommended parametric template.
