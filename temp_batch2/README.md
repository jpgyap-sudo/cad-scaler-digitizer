# HomeU Shopify Templates Batch 2

Generated from the next 50 Shopify products after Batch 1.

Use these as product-specific template seeds for the confirmation loop:
1. Vision AI analyzes uploaded product image.
2. Template matcher searches `visual_dna_index.json`.
3. App proposes the closest HomeU product template.
4. User confirms/corrects visual DNA.
5. Confirmed JSON feeds preview + DXF generator.

These are not final CAD geometry files yet. They are template memory resources.
