# Software Operator Skill

Use this for operating the app safely:
1. Upload image/PDF.
2. Choose furniture type if auto-detection confidence is below 0.75.
3. Enter at least one real dimension for scale.
4. Confirm separated views: top, front, side.
5. Run digitize.
6. Inspect warnings.
7. Export DXF.
8. Open in LibreCAD/FreeCAD/AutoCAD.
9. If dimension mismatch >2%, correct dimensions and regenerate.
