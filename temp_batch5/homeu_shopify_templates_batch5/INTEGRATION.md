# Integration

Copy this folder into:

resources/product_templates/shopify_batch_5/

Then update your registry to include:

resources/product_templates/shopify_batch_5/catalog_index.json

Important:
- Do not generate DXF directly from Shopify metadata.
- Use these files to propose a likely furniture/light/panel template.
- Let the user confirm shape, base, arms, legs, cushion layout, or panel profile.
