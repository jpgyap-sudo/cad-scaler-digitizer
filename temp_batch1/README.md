# HomeU Shopify Template Resources — Batch 1

Generated from the first 50 Shopify products pulled through the connected Shopify store.

This pack is for the **photo → furniture analysis → user confirmation → preview → DXF** loop.

Important:
- JPEG/PNG is used only as visual reference.
- The JSON template is the real guide for CAD generation.
- Do not let AI draw DXF directly.
- AI should output/confirm structure first.

Folders:
- `templates/` — one JSON per Shopify product
- `catalog_index.json` — index of products in this batch
- `INTEGRATION.md` — where to place these files in cad-scaler-digitizer

Next batch can be generated using the Shopify `endCursor` from the previous product query.
